ARNOLD QUOTES API

This is a RESTful API built on the CRUD principals.
Therefore this API supports GET, POST, PUT and DELETE.
Altough the information here only shows and supports GET,
should you want to use the API yourself, feel free to download the project on Github. You only need to set up your own MongoDB (So you don't mess up mine!), so you can GET, POST, PUT and DELETE as much as you want. The complete database will be avalible in the project folder in JSON format for insertion in your database.
All GET requests return a JSON-file.

For the complete database of quotes.

GET /api/quotes

For only one quote, apply the quote id, which is between 1 - 100

GET /api/quote/:id
